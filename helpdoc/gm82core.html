Game Maker 8.2 Core

Core, also known as gm82core, is a quality of life improvement extension for Game Maker 8.1. It adds a number of helper functions, precise timing, globalvars and utility constants, geometry and math, input, registry, and also some functions imported from studio.

Core is also required for running most of the other extensions in the GM 8.2 family.

Created by renex & Floogle. Contains contributions by Lovey01 and includes parts of High Resolution Timer 2 - check the License for usage.

Notes

  • The core object must be active at all times. It is used to run step code for the extension and any attached accessory extensions (such as gm82snd).
  • The undefined constant is used by a few functions as an error return value. It's just a string value.

Globalvars

These values are updated for you, and can be used in code.

delta_time* Number of milliseconds since last frame.
fps_fast* Fast FPS estimate based on well-known FRAPS algorithm.
fps_real* Same as Studio - measures theoretical maximum fps based on one second divided by the time it took to complete the last game step.
windows_version* Windows version as a number - 5 = XP, 6 = Vista, etc (8 and later return 8).

Utility

These functions provide shortcuts for common operations.

draw_self_floored() Draws the instance at a floored coordinate instead of fractionary.
ds_list_equal(list1,list2) Compares two ds lists to see if their contents match. Returns true or false.
ds_map_get(map,key) Gets key from map. If the key doesn't exist, undefined is returned.
ds_map_set(map,key,value) Sets key from map to value. If the key already exists, it is replaced.
ds_map_read_ini(map,filename) Windows version as a number - 5 = XP, 6 = Vista, etc (8 and later return 8).
event_draw() Calls the instance's Draw event.
event_step() Calls the instance's Step event.
instance_create_moving(x,y,object,speed,direction) Creates an instance of object at x,y moving with speed and direction. The speed and direction values are given to the instance before the Create event is executed.

Files

Functions for dealing with files.

ds_map_read_ini(map,filename) Reads ini file at filename into map. Section names are prepended to keys, separated by a space.
file_size(filename) Returns the size of file filename. If the file doesn't exist, -1 is returned.
file_text_read_all(fname,[separator]) Reads file filename in its entirety and returns it as a string. If a separator is set, it is used inbetween lines.

Colors

Functions for dealing with color values.

color_blend(col1,col2) Multiplies col1 with col2 and returns the color.
color_get_luminance(color) Returns a Kodak-corrected luminance value from color (0-255).
merge_color_corrected(col1,col2,factor) Merges col1 towards col2 by factor. Uses corrected square formula for more natural results.
color_reverse(color) Reverses the blue and red components of color (works both ways).
